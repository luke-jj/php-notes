# Slim Framework


## Create a basic mvc skeleton

    composer create-project slim/slim-skeleton mvc-framework

    set the /public/ folder as the www root for your server

## Framework Folder Structure

    logs/               log files

    public/             publically accessible files

    src/                controllers

    templates/          views

    tests/              unit tests

    vendor/             dependencies
